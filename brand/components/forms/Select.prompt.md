Native `<select>` styled to match Input — deepest sunken fill, custom chevron. Use for account-type and payment-type pickers.

```jsx
<Select label="Account type" defaultValue="credit_card">
  <option value="checking">Checking</option>
  <option value="credit_card">Credit Card</option>
  <option value="loan">Loan</option>
</Select>
```
