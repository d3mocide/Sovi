Small status pill — tinted fill + matching low-opacity border, fully rounded. Sovi's `warning` is amber (attention), never alarm red.

```jsx
<Badge variant="positive">On track</Badge>
<Badge variant="warning">Needs classification</Badge>
<Badge variant="accent">SimpleFIN</Badge>
<Badge variant="muted">Credit Card</Badge>
```

Variants: `default`, `accent` (sky), `positive` (emerald), `warning` (amber), `muted`. Keep label text to 1–3 words; pass an icon node as a child for a leading glyph.
